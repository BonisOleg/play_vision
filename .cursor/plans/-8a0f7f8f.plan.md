# Виправлення стрілочок Featured Courses

## Проблеми

### 1. home-additions.css (Featured Variant A)

Стрілочки залишаються зліва/справа через:

```css
.featured-carousel-container {
    padding: 0 40px;  /* <- тут стрілочки */
}
```

Треба змінити на позиціонування знизу як в інших каруселях.

### 2. featured-courses-light.css (Featured Variant B)

Є `display: contents` та `order`, але НЕМАЄ `display: flex` для батьківського контейнера, тому `order` не працює.

## Виправлення

### Файл: static/css/components/home-additions.css

У блоці `@media (max-width: 767px)` (рядок 312):

**ЗАМІНИТИ:**

```css
.featured-carousel-container {
    padding: 0 40px;
}

.carousel-btn {
    width: 36px;
    height: 36px;
    min-width: 36px;
    min-height: 36px;
}

.carousel-btn svg {
    max-width: 18px;
    max-height: 18px;
}
```

**НА:**

```css
.featured-courses-header {
    display: contents;
}

.featured-courses-header .section-title {
    order: 1;
    margin-bottom: 16px;
}

.featured-carousel-container {
    order: 2;
    padding: 0 12px;
}

.carousel-navigation {
    order: 3;
    align-self: center;
    width: fit-content;
    padding-top: 20px;
}

.carousel-btn {
    width: 36px;
    height: 36px;
    min-width: 36px;
    min-height: 36px;
}

.carousel-btn svg {
    width: 18px;
    height: 18px;
}
```

### Файл: static/css/components/featured-courses-light.css

У блоці `@media (max-width: 767px)` (рядок 300) **ДОДАТИ НА ПОЧАТКУ:**

```css
.featured-courses-section .section-content {
    display: flex;
    flex-direction: column;
}
```

Щоб `order` працював для `.featured-courses-header`, `.featured-carousel-container` та `.featured-navigation`.

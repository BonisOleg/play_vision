# План: Стильні блоки в стилі карток підписки

## Мета

Розділити блоки "Для кого" та "Що ти отримуєш" на два окремі стильні блоки в стилі карток підписки - чисто, елегантно, мінімалістично.

## Зміни

### 1. HTML структура

**Файл:** `templates/events/event_detail.html` (рядки 63-87)

Замінити об'єднаний `.event-benefits-section` на два окремі блоки:

- Контейнер `.event-info-cards` з grid layout
- Блок "Для кого" - окрема картка `.event-info-card`
- Блок "Що ти отримуєш" - окрема картка `.event-info-card`
- Кожен блок має свій бордер, тінь, padding

### 2. CSS стилізація

**Файл:** `static/css/components/events.css` (рядки 1236-1275)

Створити стилі в стилі карток підписки:

**Загальні стилі карток:**

- Білий фон (або `var(--color-bg)`)
- Тонкий сірий бордер `1px solid #e5e5e5`
- М'яка тінь `box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08)`
- Заокруглені кути `border-radius: 12px`
- Padding `32px`
- Hover ефект: підняття + більша тінь
- Smooth transition для всіх ефектів

**Grid layout:**

- Дві колонки на десктопі `grid-template-columns: repeat(2, 1fr)`
- Одна колонка на мобільних
- Gap між картками `24px`

**Типографіка:**

- Заголовок: напівжирний, темний, з відступом
- Список: без bullets, з кастомними іконками (червоні кружечки)
- Відступи між елементами

**Темна тема:**

- Темний фон
- Світлий текст
- Бордер rgba для прозорості

**Адаптивність:**

- < 768px: одна колонка, менший padding

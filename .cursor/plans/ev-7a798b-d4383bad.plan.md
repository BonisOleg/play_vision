# Виправлення конфліктів CSS для карток експертів

## Проблема

Картки експертів на сторінці івенту виглядають неправильно через:

1. Дублікат `.experts-grid` в `mentoring.css` (рядки 229-235) перезаписує стилі з `expert-cards-base.css`
2. Неправильний порядок підключення CSS файлів - `events.css` завантажується після базових стилів карток

## Рішення

### 1. Видалити дублікат `.experts-grid` з mentoring.css

**Файл:** `static/css/components/mentoring.css`

Видалити рядки 229-235:

```css
.experts-grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 24px;
    margin: 48px auto 0;
    max-width: 1200px;
}
```

Цей блок дублює стилі з `expert-cards-base.css` і не потрібен, бо базові стилі вже є.

### 2. Змінити порядок підключення CSS в event_detail.html

**Файл:** `templates/events/event_detail.html`

Змінити блок `{% block extra_css %}` з:

```html
{% block extra_css %}
<link rel="stylesheet" href="{% static 'css/components/section-base.css' %}">
<link rel="stylesheet" href="{% static 'css/components/unified-sections.css' %}">
<link rel="stylesheet" href="{% static 'css/components/events.css' %}">
<link rel="stylesheet" href="{% static 'css/components/expert-cards-base.css' %}">
<link rel="stylesheet" href="{% static 'css/components/expert-flip-cards.css' %}">
{% endblock %}
```

На:

```html
{% block extra_css %}
<link rel="stylesheet" href="{% static 'css/components/section-base.css' %}">
<link rel="stylesheet" href="{% static 'css/components/unified-sections.css' %}">
<link rel="stylesheet" href="{% static 'css/components/expert-cards-base.css' %}">
<link rel="stylesheet" href="{% static 'css/components/expert-flip-cards.css' %}">
<link rel="stylesheet" href="{% static 'css/components/events.css' %}">
{% endblock %}
```

**Чому:** Як на home.html та mentoring.html - expert-cards завантажуються перед специфічними стилями сторінки, щоб специфічні стилі могли перезаписати базові при потребі.

### 3. Додати стилі для кнопки "Детальніше" в expert-flip-cards.css

**Файл:** `static/css/components/expert-flip-cards.css`

Після рядка 102 (після `.expert-detail-link:hover`) додати стилі для `.expert-detail-btn`:

```css
/* КНОПКА "ДЕТАЛЬНІШЕ" */
.expert-detail-btn {
    display: block;
    width: 100%;
    background: none;
    border: none;
    color: #d32f2f;
    font-size: 0.9375rem;
    font-weight: 500;
    text-decoration: underline;
    text-align: center;
    cursor: pointer;
    transition: color 0.3s ease, text-decoration-color 0.3s ease;
    margin-top: auto;
    padding: 4px 0;
    font-family: inherit;
}

.expert-detail-btn:hover {
    color: #b71c1c;
    text-decoration-color: #b71c1c;
}
```

**Чому:** В HTML використовується `<button class="expert-detail-btn">`, але стилів для цього класу немає. Є тільки стилі для `.expert-detail-link` (який не використовується). Потрібно додати ідентичні стилі для кнопки.

## Результат

Після цих змін картки експертів на сторінці івенту будуть виглядати ідентично до карток на головній та ментор-коучинг сторінках.

# Додавання border для карток в світлій темі

## Мета

Додати чорний border для карток експертів в світлій темі, використовуючи наявний код без дублів та конфліктів.

## Аналіз поточного стану

### Базові стилі (світла тема)

**Файл:** `static/css/components/expert-flip-cards.css` (рядки 39-52)

- `.expert-card-front, .expert-card-back` - НЕ мають border

### Темна тема

**Файл:** `static/css/components/expert-flip-cards.css` (рядки 320-325, 350-355)

- Має `border: 1px solid var(--color-border);` (білий)

### CSS змінні

- Світла тема: `--color-border: #000000;` (чорний)
- Темна тема: `--color-border: #ffffff;` (білий)

## План виправлення

### Крок 1: Додати border в базові стилі

**Файл:** `static/css/components/expert-flip-cards.css` (рядки 39-52)

**Додати:** `border: 1px solid var(--color-border);` до `.expert-card-front, .expert-card-back`

**Результат:** Border буде чорним в світлій темі (через `--color-border: #000000`)

### Крок 2: Видалити дублікат border з темної теми

**Файл:** `static/css/components/expert-flip-cards.css` (рядки 320-325, 350-355)

**Видалити:** `border: 1px solid var(--color-border);` з обох блоків темної теми

**Причина:** Border буде наслідуватися з базових стилів, а `--color-border` в темній темі вже білий

## Результат

- ✅ Світла тема: чорний border (через базові стилі + `--color-border: #000000`)
- ✅ Темна тема: білий border (через базові стилі + `--color-border: #ffffff`)
- ✅ Немає дублів - border визначений один раз в базових стилях
- ✅ Немає конфліктів - CSS змінні автоматично змінюють колір

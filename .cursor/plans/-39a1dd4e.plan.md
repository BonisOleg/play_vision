# План: Обмеження ширини та розміщення кнопки з точками на мобільних

## Частина 1: Обмеження ширини заголовка та підзаголовка

### Перевірка логічності та безпеки

✅ **Логічність:**

- `max-width: 70%` для заголовка - обмежує ширину до 70% від батьківського блоку (`.section-content.hero-new-design`)
- `max-width: 50%` для підзаголовка - обмежує ширину до 50% від батьківського блоку
- Це забезпечує перенос тексту на нові рядки, покращуючи читабельність на мобільних

✅ **Крос-платформенність:**

- `max-width` з відсотками працює однаково на всіх браузерах та пристроях
- Не використовуємо специфічні властивості, які можуть не підтримуватися

✅ **Безпека:**

- Стилі застосовуються тільки в `@media (max-width: 768px)` - не впливає на desktop/tablet
- Селектор `.hero-new-design .hero-title` специфічний - не впливає на інші hero секції
- Додаємо тільки `max-width`, не змінюємо інші властивості (font-size, margin, line-height залишаються)

### Зміни

**Файл:** [static/css/components/home.css](static/css/components/home.css)

**Рядок:** 105-115 (всередині `@media (max-width: 768px)`)

**Додати до існуючих стилів:**

```css
.hero-new-design .hero-title {
    font-size: 1.5rem;
    margin-bottom: -1rem;
    line-height: 1.2;
    max-width: 70%; /* ← ДОДАТИ */
}

.hero-new-design .hero-subtitle {
    font-size: 0.875rem;
    margin-bottom: 0.3rem;
    line-height: 1.4;
    max-width: 50%; /* ← ДОДАТИ */
}
```

## Частина 2: Кнопка та точки в одному рядку на мобільних

### Завдання

На мобільних версіях (≤768px):

- Кнопка (`.hero-buttons`) та точки слайдера (`.hero-slider-dots`) мають бути в одному рядку
- Кнопка зліва, точки справа
- Весь контент опустити нижче, щоб кнопка та точки були внизу екрану

### Зміни

**Файл:** [static/css/components/home.css](static/css/components/home.css)

**Рядок:** 117-148 (всередині `@media (max-width: 768px)`)

**1. Об'єднати кнопку та точки в один контейнер через flexbox:**

**Замінити:**

```css
.hero-new-design .hero-buttons {
    gap: 0.5rem;
    margin-bottom: 0.5rem;
}

/* ... інші стилі ... */

.hero-slider-dots {
    margin-top: 0.5rem;
    gap: 5px;
}
```

**На:**

```css
/* Об'єднуємо кнопку та точки в один рядок */
.hero-new-design .hero-buttons {
    gap: 0.5rem;
    margin-bottom: 0;
    display: flex;
    align-items: center;
}

/* Точки справа від кнопки */
.hero-new-design .hero-slider-dots {
    margin-top: 0;
    margin-left: auto; /* Притискаємо точки справа */
    display: flex;
    align-items: center;
    gap: 5px;
}

/* Обгортка для кнопки та точок в одному рядку */
.hero-new-design .hero-buttons + .hero-slider-dots {
    display: flex;
    width: 100%;
    justify-content: space-between;
    align-items: center;
    margin-top: 0.5rem;
}
```

**АБО простіше - обгорнути обидва елементи:**

**Альтернативне рішення (якщо HTML структура дозволяє):**

Додати стилі для обгортки кнопки та точок:

```css
/* Контейнер для кнопки та точок в одному рядку */
.hero-new-design .hero-buttons {
    display: flex;
    align-items: center;
    justify-content: space-between;
    width: 100%;
    margin-bottom: 0;
    gap: 0.5rem;
}

/* Точки всередині того ж контейнера або поруч */
.hero-new-design .hero-slider-dots {
    margin-top: 0;
    margin-left: auto;
    display: flex;
    align-items: center;
    gap: 5px;
}
```

**2. Опустити контент нижче:**

Оскільки `.section-content.hero-new-design` вже має `justify-content: flex-end`, контент вже притиснутий до низу. Можливо потрібно збільшити `padding-bottom` або додати `margin-top: auto` для заголовка/підзаголовка.

**Додати:**

```css
.hero-new-design .hero-title {
    /* ... існуючі стилі ... */
    margin-top: auto; /* Притискає контент до низу */
}
```

**АБО збільшити padding-bottom:**

```css
.hero-section .section-content.hero-new-design {
    /* ... існуючі стилі ... */
    padding: 0.75rem 1rem 1rem 0; /* Збільшити padding-bottom з 0.5rem на 1rem */
}
```

## Безпека змін

✅ **Не впливає на:**

- Desktop/Tablet (>768px) - всі зміни всередині `@media (max-width: 768px)`
- Інші hero секції - селектори специфічні для `.hero-new-design`

✅ **Каскад:**

- Стилі додаються до існуючих правил
- `margin-left: auto` для точок притискає їх справа
- `justify-content: space-between` розподіляє кнопку та точки по краях

## Результат

- **Mobile (≤768px):**
  - Заголовок: max-width 70%, перенос на новий рядок
  - Підзаголовок: max-width 50%, перенос на новий рядок
  - Кнопка та точки в одному рядку: кнопка зліва, точки справа
  - Весь контент притиснутий до низу екрану
- **Tablet/Desktop:** без змін

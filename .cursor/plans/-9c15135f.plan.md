# Виправлення кнопки Hero та підзаголовка ментор коучингу

## Проблеми

1. Кнопка Hero стає червоною в світлій темі через загальний стиль `.btn-primary` з `main.css`
2. Потрібно уніфікувати мобільний дизайн кнопки Hero (білий фон) для світлої та темної теми
3. Підзаголовок в ментор коучингу потрібно відцентрувати по лівій стороні та прибрати зайві відступи

## Рішення

### 1. Виключити `.hero-btn-new` з червоних стилів в `main.css`

**Файл:** `static/css/main.css`

Змінити `.btn-primary` на `.btn-primary:not(.hero-btn-new)` (рядки 165-172), щоб кнопка Hero не ставала червоною в світлій темі.

### 2. Додати білий фон для мобільних версій кнопки Hero в світлій темі

**Файл:** `static/css/components/home.css`

Додати `background: #ffffff` до мобільних стилів `.hero-btn-new` (рядок 178-185), щоб уніфікувати дизайн з темною темою.

### 3. Відцентрувати підзаголовок ментор коучингу по лівій стороні та прибрати зайві відступи

**Файл:** `static/css/components/unified-sections.css`

Оновити стилі для `.ecosystem-subtitle` (рядок 130-133):

- Додати `text-align: left;`
- Прибрати або зменшити `margin-bottom` якщо є зайві відступи

Також перевірити стилі в `home.css` для `.mentor-coaching-section .ecosystem-subtitle` (рядок 719).

## Результат

- Кнопка Hero біла в обох темах на всіх пристроях
- Мобільний дизайн кнопки уніфікований
- Підзаголовок ментор коучингу вирівняний по лівій стороні без зайвих відступів

# Біла кнопка Hero та виправлення кнопки ментор коучингу

## Проблеми

1. Кнопка Hero може ставати червоною в темній темі через загальні стилі `.btn-primary` з `theme.css`, а також має сірий фон замість білого
2. Кнопка "Детальніше" в ментор коучингу на мобільних версіях в темній темі не змінює стиль - залишається темною з темним шрифтом через `!important` в базовому стилі

## Рішення

### 1. Виключити `.hero-btn-new` з червоних стилів в `theme.css`

**Файл:** `static/css/theme.css`

Додати `:not(.hero-btn-new)` до правил для `.btn-primary` в темній темі (рядки 43-52).

### 2. Змінити фон кнопки Hero на білий в базових стилях

**Файл:** `static/css/components/home.css`

Змінити `background: #e9e9e9` на `background: #ffffff` (білий) для `.hero-btn-new`:

- Рядок 277: Desktop стилі (≥768px)
- Рядок 314: Hover ефект (змінити на `#f5f5f5`)

### 3. Оновити стилі для темної теми Hero кнопки

**Файл:** `static/css/components/home.css`

Змінити `background: #e9e9e9` на `background: #ffffff` в правилах темної теми:

- Рядок 672: `[data-theme="dark"] .hero-btn-new`
- Рядок 677: `[data-theme="dark"] .hero-btn-new:hover`
- Рядок 816: `[data-theme="auto"] .hero-btn-new`
- Рядок 821: `[data-theme="auto"] .hero-btn-new:hover`

### 4. Виправити кнопку ментор коучингу для мобільних в темній темі

**Файл:** `static/css/components/home-additions.css`

**4.1. Прибрати `!important` з базового стилю:**

- Рядок 233: змінити `background: #171717 !important;` на `background: #171717;`
- Рядок 239: змінити `border-radius: 50px !important;` на `border-radius: 50px;`

**4.2. Додати медіа-запити для мобільних версій в темній темі:**

Після рядка 422 додати медіа-запити з `!important` для мобільних версій, щоб перекрити базовий стиль.

## Результат

- Кнопка Hero завжди біла (`#ffffff`) з чорним текстом на всіх пристроях та темах
- Кнопка "Детальніше" в ментор коучингу правильно змінює стиль на мобільних в темній темі
- Обидві кнопки працюють коректно на desktop, tablet та mobile

# Виправлення сітки відео на мобільних

## Проблема

На скріншотах видно, що `.events-video-grid` відображається як вузька вертикальна смуга замість повноцінної grid сітки.

## Причина

У рядку 89 є `margin-right: 20px` для `.events-video-grid`, що на мобільних у поєднанні з `padding: 10px` батьківського контейнера може обрізати або стискати сітку.

## Рішення

### Зміна 1: Прибрати margin-right на мобільних

У файлі `static/css/components/events.css` в блоці `@media (max-width: 768px)` (після рядка 699) додати:

```css
.events-video-grid {
    grid-template-columns: repeat(3, 1fr);
    gap: 8px;
    margin-right: 0;
    margin-left: 0;
    width: 100%;
}
```

**Обґрунтування:**

- `margin-right: 0` прибирає відступ справа, що звільняє місце
- `margin-left: 0` гарантує відсутність відступів зліва
- `width: 100%` гарантує, що сітка займе всю доступну ширину

### Зміна 2: Переконатися що gap правильний

Gap вже змінено з `5px` (десктоп) на `8px` (мобільні), це добре для відображення.

### Зміна 3: Перевірити мінімальну висоту

`min-height: 100px` є правильним для 3 колонок на мобільних.

## Крос-платформенність

✅ **iOS Safari**: `grid-template-columns`, `margin`, `width` підтримуються з iOS 10.1+

✅ **Android Chrome**: Підтримується з Android 5.0+

✅ **Всі браузери**: Стандартні CSS властивості

## Очікуваний результат

Сітка відео займатиме всю ширину екрану (з урахуванням padding контейнера) і покаже всі 9 елементів у нерегулярній структурі (великі займають 2 колонки, малі - 1 колонку) в 3-колонковій сітці.

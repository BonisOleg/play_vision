# Зменшення margin-bottom для events-intro-subtitle на мобільних

## Проблема

Селектор `.events-intro-subtitle` має `margin-bottom: 100px` (рядок 58), що занадто багато для мобільних версій.

## Рішення

### Файл: [static/css/components/events.css](static/css/components/events.css)

### Зміна: Додати правило в мобільний медіа-запит (після рядка 725)

В існуючому блоці `@media (max-width: 768px)` після `.events-categories-grid` додати:

```css
.events-intro-subtitle {
    margin-bottom: 10px;
}
```

## Результат

На мобільних версіях (max-width: 768px) `.events-intro-subtitle` матиме `margin-bottom: 10px` замість 100px, що зменшить відстань між підзаголовком та категоріями.

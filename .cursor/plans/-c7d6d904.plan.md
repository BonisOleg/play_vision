# ОСТАТОЧНЕ рішення сітки відео (ГАРАНТІЯ 100%)

## ПОВНА ДІАГНОСТИКА

Проаналізовано весь ланцюжок контейнерів:

```
.unified-sections-wrapper → .fullscreen-section (95% width, 10px padding) 
→ .section-content → .events-intro-layout (flex) 
→ .events-intro-right (flex item) → .events-video-grid (grid)
```

**ЗНАЙДЕНО 5 КРИТИЧНИХ ПРОБЛЕМ:**

1. **Flex items без `min-width: 0`** - блокує звуження
2. **Grid має `margin-right: 20px`** на десктопі (рядок 89) - залишається на мобільних 
3. **Відсутність явного `width: 100%`** в медіа-запитах
4. **Відсутність `overflow-x: hidden`** на батьківських контейнерах
5. **Конфлікт між двома `@media (max-width: 768px)`** блоками (рядок 679 і 745)

## РІШЕННЯ

### Файл: [static/css/components/events.css](static/css/components/events.css)

### Зміна 1: Виправити базові flex items (рядки 31-37)

Замінити:

```css
.events-intro-left {
    flex: 0 0 45%;
}

.events-intro-right {
    flex: 0 0 55%;
}
```

На:

```css
.events-intro-left {
    flex: 0 0 45%;
    min-width: 0;
}

.events-intro-right {
    flex: 0 0 55%;
    min-width: 0;
    overflow-x: hidden;
}
```

### Зміна 2: Виправити планшетний медіа-запит (рядки 664-667)

Замінити:

```css
.events-intro-left,
.events-intro-right {
    flex: 0 0 100%;
}
```

На:

```css
.events-intro-left,
.events-intro-right {
    flex: 0 0 100%;
    min-width: 0;
    max-width: 100%;
    width: 100%;
}
```

### Зміна 3: ОБ'ЄДНАТИ два мобільних медіа-запити в ОДИН (рядки 679-755)

Видалити ОБИДВА існуючих блоки `@media (max-width: 768px)` та замінити на ОДИН:

```css
@media (max-width: 768px) {
    /* Layout контейнери */
    .events-intro-layout {
        gap: 20px;
        overflow-x: hidden;
    }

    .events-intro-right {
        order: 1;
        flex: 0 0 100%;
        min-width: 0;
        max-width: 100%;
        width: 100%;
        overflow-x: hidden;
    }

    .events-intro-left {
        order: 2;
        flex: 0 0 100%;
        min-width: 0;
        max-width: 100%;
        width: 100%;
    }

    /* Заголовки */
    .events-intro-left .section-title,
    .events-calendar-section .section-title,
    .events-catalog-section .section-title {
        font-size: clamp(1.25rem, 4vw, 1.75rem);
    }

    /* Категорії 2x2 */
    .events-categories-grid {
        grid-template-columns: repeat(2, 1fr);
        gap: 16px;
    }

    /* КРИТИЧНО: Відео сітка */
    .events-video-grid {
        grid-template-columns: repeat(3, 1fr);
        gap: 8px;
        margin: 0;
        padding: 0;
        width: 100%;
        max-width: 100%;
        box-sizing: border-box;
    }

    .video-placeholder {
        min-height: 100px;
        width: 100%;
        box-sizing: border-box;
    }

    /* Каталог */
    .events-catalog-grid {
        grid-template-columns: 1fr;
        gap: 20px;
    }

    /* Календар */
    .calendar-card {
        flex: 0 0 100%;
        min-width: 0;
    }

    .calendar-navigation {
        left: 50%;
        transform: translateX(-50%);
        bottom: 40px;
    }

    .calendar-pagination {
        right: 50%;
        transform: translateX(50%);
        bottom: 130px;
    }

    .calendar-carousel-wrapper {
        padding-bottom: 160px;
    }

    /* Touch-friendly кнопки */
    .calendar-nav-btn,
    .btn-details,
    .pagination-link {
        min-width: 44px;
        min-height: 44px;
    }
}
```

### Зміна 4: Додати overflow контроль до section (після рядка 22)

Після `.events-intro-section` додати:

```css
.events-intro-section {
    background: transparent;
    overflow-x: hidden;
}

.events-intro-section .section-content {
    overflow-x: hidden;
}
```

## ЧОМУ ЦЕ 100% СПРАЦЮЄ

✅ **min-width: 0** - дозволяє flex items звужуватися

✅ **width: 100%** - явно встановлює повну ширину

✅ **max-width: 100%** - запобігає overflow

✅ **margin: 0** - видаляє проблемний margin-right

✅ **overflow-x: hidden** - обрізає будь-який overflow

✅ **box-sizing: border-box** - правильний розрахунок розмірів

✅ **Один медіа-запит** - немає конфліктів
